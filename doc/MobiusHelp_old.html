Mobius

Visual Dataflow Modelling Tool for Designers

What is a Mobius Data Object?

The MobiusDataObject is the internal datastructure used by Mobius. It has the following public properites and methods:

  • geometry : Stores the underlying geometry of the object - could be normal geometry or NURBS geometry

  • geometryUpdated: Stores a boolean as status of the geometry - set to true of the geometry has been updated.

  • extractGeometry() : Stores the visual representation of the object as a mesh

  • extractData() : Stores the data associated with the object

    • Geometry

    Making Geometry

    • makePositionVector(x, y, z)

      Input: x, y, z coordinates

      Output: MobiusDataObject with Geometry

      Description: Creates a vector to a position in space

      Example: makePositionVector( 10, 10, 10)

    • makePositionVectorsFromPoints ( list_of_points )

      Input: List of x, y, z coordinates

      Output: List of MobiusDataObjects with Geometry

      Description: Creates a list of vectors to a position in space

      Example: makePositionVectorsFromPoints([12,0,0], [1,23,0], [12, 5, 3])

    • makeCircle (radius, segments)

      Input: Radius and number of segments

      Output: MobiusDataObject with Circle geometry

      Description: Creates a circular shape; Fine-ness depends on the number of segments.

      Example: makeCircle (12, 30)

    • makePolyline (arrOfPoints)

      Input: List of points

      Output: MobiusDataObject with Polyline Geometry

      Description: Creates a polyline from a list of 3D points

      Example: makePolyline ([[0,0,0], [10,0,0], [10,0,10], [7,0,15], [3,0,6]])

    • makeBox (length, breadth, height)

      Input: Dimensions of the box

      Output: MobiusDataObject with Box geometry

      Description: Creates a box at the origin with the given dimensions

      Example: makeBox (12, 12, 12)

    • makeSphere (radius)

      Input: Radius of the sphere

      Output: MobiusDataObject with Sphere geometry

      Description: Creates a sphere at the origin with the given radius

      Example: makeSphere (15)

    • makePolygonByPoints ( pointsXY )

      Input: 2D coordinates of the polygon. The x Direction is outward from the screen; the y direction points upwards. Points have to specified anticlockwise

      Output: MobiusDataObject with flat Shape Geometry

      Description: Creates a MobiusDataObject with 2D polygon which can be passed further to extrude function under geometry to get a solid 3D object

      Example: makePolygonByPoints ( pointsXY )

    • extrudePolygon (mObj, thickness, bevel )

      Input: MobiusDataObject with 2D Polygon Shape

      Output: MobiusDataObject with Geometry

      Description: Extrudes a polygon in a particular direction; Currently, extrusion in only z-direction is supported

      Example: extrudePolygon( mobiusDataObject, 12, true)
